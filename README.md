# Proyecto Análisis Climático UTN

**Integrantes:** Hugo, Paco y Luis.
**Escenario:** A - Análisis de Datos Climáticos.

## Descripción del Dataset
El proyecto utiliza datos de anomalías de temperatura global. El archivo de datos no se incluye directamente en este repositorio para optimizar el almacenamiento, cumpliendo con las buenas prácticas de versionado.

## Instrucciones para ejecutar el script
1. Descargar el dataset desde [Datahub Global Temp](https://datahub.io/core/global-temp/r/monthly.csv).
2. Guardar el archivo descargado con el nombre `monthly.csv` estrictamente dentro de la carpeta `/datos`.
3. Ejecutar el script principal desde la raíz del proyecto usando el comando: `python scripts/analisis_climatico.py`.
4. Los gráficos estadísticos se generarán automáticamente en la carpeta `/resultados`.
